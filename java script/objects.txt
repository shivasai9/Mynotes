 an object in java script is nothing but a collection of name value pairs

  name value pair is a object

ex:
    address(name) = 'xyz'(value) // name value pair

value may be another name value pair or another object

ex:
   address = { 
              name = "shiva sai",
              street = { 
                        no = 2434,
                        name = 'xyz'
                        }

               }
